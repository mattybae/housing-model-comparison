# housing-model-comparison
Model comparison (Scaled Linear Regression vs. Random Forests) on California housing dataset using mean-absolute-error to evaluate.

This repository contains a Jupyter notebook that compares Linear Regression & Random Forest on the California housing dataset
from *Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow*.

## File

- `model_comparison_housing_dataset.ipynb`: End-to-end EDA and model comparison using mean-absolute-error.
